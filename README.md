#  Web Scraping and Data Analysis of Online Book Store using Python

## Description
This project scrapes book data from “Books to Scrape” using Python’s BeautifulSoup. It extracts titles, prices, ratings, and availability, cleans them using Regex and pandas, handles missing values, and visualizes insights with Matplotlib and Seaborn. Demonstrates end-to-end web scraping and data analysis.

---

## Objectives
- Extract book information from multiple pages (title, price, rating, availability).
- Clean and preprocess scraped data using **Regex** and **Pandas**.
- Handle missing values and inconsistencies.
- Perform data visualization using **Matplotlib** and **Seaborn**.
- Export cleaned data into a structured CSV file.

---

## Project Flow
1. **Web Scraping** – Extract book details from multiple pages using `requests` and `BeautifulSoup`.
2. **Data Cleaning** – Use Regex to extract numeric values, clean text, and fix missing data.
3. **Exploratory Data Analysis** – Visualize data distributions, correlations, and insights.
4. **Data Export** – Save cleaned data to `books_data_cleaned.csv` for reuse.
